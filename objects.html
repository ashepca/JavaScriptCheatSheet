1.6 JS Object General Overview

Javascript is an object oriented programming language. Objects are things in javascript which have properties and methods, which allow you to store and manipulate more complicated data, verses the primitive data types of variables. Objects must be created, then you can use the objects properties and methods.


General Syntax

To create an object, you can use the new operator with the object constructor (which is a function that describes how the object is created), like this:

var person1 = new Object(value, value);

Or, you can write it like this:

var person1 = {};

Or, you can use the literal syntax instead of calling the Object constructor, like this:

person1.firstName = "Name";

person1.lastName = "Surname";


Methods

Methods are actions that can act upon objects which are stored as in properties as functions. To access an Objects Methods you write it like this:

objectName.methodName()


Properties

In Javascript, the name: "value" pairs of data within an object are properties. The name is the property and the value is the property value. To access an Objects properties you write it like this:

objectName.propertyName

OR

objectName["propertyName"]


Examples

Here is an example of Javascript code:

//script example for 1.6 JS Object General Overview

function showExample() {

alert("Let's create a person object");

var person = {};

person.firstName = prompt("Please enter a first name:");

person.lastName = prompt("Please enter a last name:");

person.age = prompt("Please enter an age:");

alert("You have created a person object with the following properties:\nFirst Name: " + person.firstName + "\nLast Name: " + person.lastName + "\nAge: " + person.age);

}
