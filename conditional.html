1.3 Conditional Statements

Conditional Statements in Javascript allow your program to make decisions based on whether a specific condition has been met, and then execute only the appropriate portion of the program. Conditional statements include the if statement, the if else statement and the switch statement.


General Syntax

The if statement is written as:

if (test condition) {

executed code if test condition evaluates to true;

}

The test condition uses a comparison operator, such as 4 > 2 or variable x <= y. Alternatively, an else statement may be added which is executed whenever the condition is false, written as:

if (test condition) {

executed code if test condition evaluates to true;

} else {

executed code if test condition is false;

}

The else if statement adds additional test conditions, such as:

if (test condition) {

executed code if test condition evaluates to true;

} else if (second test condition) {

executed code if second test condition is true;

} else {

executed code if no test conditions are true;

}

The switch statement is a more efficient method of checking multiple test conditions if there are many tests. It is written like:

switch(variable to check) {

case "one":

executed code;

break;

case: "two"

executed code;

break;

default:

executed code;

}


Methods

The break key word ends the switch statement and moves to the closing { bracket.

The default case is used if the value does not match any of the declared cases.

Properties

Conditional Statements do not have properties.


Examples

Here is an example of Javascript code:

//script example for 1.3. Conditional Statements

function showExample() {

alert("Let's use a conditional statement to have the program respond appropriately to your choice.");

var choice = prompt("Do you prefer jam or liver?");

switch(choice) {

case "jam":

alert("Yummy, I also prefer jam!");

break;

case "liver":

alert("Yuck, liver tastes awful!");

break;

default:

alert("Sorry, you must choose jam or liver.");

}

}
