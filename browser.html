2.1 Browser Objects

Just like Javascript, web browsers have built-in Browser Objects that are made available to save time in coding. There are many objects, including the window object, which represents the browsers frame that contains the web page, the history object which tracks the users pages visited, the location object which contains data about the current page's location.


General Syntax

The window object is a global object, which means you do not have to type it's name specifically, so these two lines are equal:

window.alert("Hello");

alert("Hello");

Otherwise, referencing browser objects is the same as all other objects.


Methods

The window object has many methods including the prompt() method which provides a pop up window asking the user to enter data. Another common method is the alert() method which provides a pop up window advising the user of some information.

The history object has methods like the go() method, which takes a parameter to specify how far forward or back in the history to go.

The location object has methods like the replace() method, which replaces the current page displayed with a new page.


Properties

The window object has many properties like .closed which returns a boolean value of whether the window has been closed or not, and .docment which returns the document object for the window.

The history object has one property .length which returns the number of URLs in the history.

The location object has an .href property which allows you to add the currently loaded page to history.
