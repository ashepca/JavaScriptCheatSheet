1.1 Variables

In Javascript, a variable is a reserved place in computer memory where data may be stored. Variables are given unique names which can be referenced to facilitate use of the data within a program. Data stored in a variable may be of several different types, such as letters, numbers, expressions, objects or functions. Variables also have scope and may be available locally or globally within a program.


General Syntax

A variable must first be created. It then may have a value assigned, which may be done at the time of creation or later. Also, the value assigned to a variable may change throughtout the program. To create a variable, begin with the keyword 'var', followed by the name of the variable. Such as:

var apple;

Note: Variable names must follow these rules:

  • they must start with a letter, $ or _
  • they are case sensitive
  • they cannot be any of the reserved javascript keywords, for example 'var'

Methods

When a variable is created, but does not have a value assigned, the default value is 'undefined'. To assign a value to a variable, use the assignment operator '=' followed by the value. Such as:

var apple = "red";

OR

var banana = 3;

Variables may also be assigned values that use other variables in an expression, such as:

var lemon = banana + 5;


Properties

Variables have a scope property which determines where within a program a variable may be accessed. A variable declared within a function has a 'local' scope and may only be accessed within the function. A variable declared outside a function has a 'global' scope and may be accessed anywhere in the program.
