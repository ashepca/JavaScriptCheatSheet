1.9 String Object

A Javascript String is a string or series of characters, including letters and numbers and special characters, or in otherwords any text contained within single or double quotes.


General Syntax

String objects can be created like any other object, such as:

var aString = new String("This is a string.");

var anotherString = new String("John");

var yetAnotherString = new String("ABC123");

But strings can also be created like other primitive variables and Javascript will convert them to string objects, like this:

var aString = "This is a string.";

var anotherString = "John";

var yetAnotherString = "ABC123"


Methods

The string object has many methods. One method is the substr() to parse out a section of a string which takes two parameters, the first is the starting index position to cut and the second is the length of the substring to remove. It is written like this:

var someString = "This is a string";

var piece = someString.substr(0, 4);

The value of piece will be "This".

Another method is the charAt() method which will tell you about a single character within a string, written like this:

var someString = "This is a string";

var char = someString.charAt(8);

The value of char will be "a".


Properties

The string object has many properties. The .length property will return the length of the string, i.e. the number of characters. The .prototype property allows you to add new properties to the string object. The .constructor property will return the string's constructor function.


Examples

Here is an example of Javascript code:

//script example for 1.9. String Object

function showExample() {

alert("Let's use some string object properties and methods on a string.");

var yourString = prompt("Write a string of any kind:");

alert("This is your string: " + yourString);

var lengthOfYourString = yourString.length;

alert("Your string is " + lengthOfYourString + " characters long");

var capitalYourString = yourString.toUpperCase();

alert("This is your string in all capitals: " + capitalYourString);

var lowerYourString = yourString.toLowerCase();

alert("This is your string in all lower case: " + lowerYourString);

}
