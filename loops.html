1.4 Loops

In Javascript, Loops are used to repeat blocks of code instead of having to retype the same code. The looped part of the code will continue to be repeated until the test condition is no longer true. There are different types of Loops, including the for loop, the for ... in loop, the while loop and the do ... while loop.


General Syntax

The for loop is written like this:

for(initialize the loop variable; loop test condition; increment the loop variable) {

executable code within the loop;

}

The while loop is written like this:

while(test condition){

executable code while test condition is true

}


Methods

Infinte loops may occur if the code is written in such a way that the test condition will always be true. Infinite loops may crash your browser. To exit an infinite loop you must shut down the program.

Break statements within if statements may be used in loops to exit the loop and proceed to the closing curly { bracket, even if the test condition of the outer loop were still true.

Conntinue statements stop the loop from executing at that point, but they return to the top of the loop and the test condition, instead of exiting the loop altogether.


Properties

Loops do not have properties.


Examples

Here is an example of Javascript code:

alert("Let's use a loop to count down.");

var cycles = parseInt(prompt("Pick a positive number:"));
