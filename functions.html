1.5 Functions

Javascript functions are blocks of code that performs a particular task by takeing information, processing it and producing a result. Functions are efficient ways of writing code, as they can be reused over and over, even in different programs. Some functions are built into Javascript, others you write yourself.


General Syntax

Functions are written like this:

function functionName(function parameters) {

executable code within the function;

return variable;

}


Methods

Values are passed into functions by listing the parameters within the round brackets (). These values are then available to be used within the function. Values do not have to be passed to a function, however, the parameters passed much match the parameters in the function as written.

To call a function, or invoke it, within a program, the function name is written with round brackets () with or without parameter values as required. This may be used to simply xecute a block of code, such as:

myFunction(3);

Or, a function may be used to assign a returned value to a variable, such as:

var pyjamas = myFunction(red, fuzzy);

Return statements are optional. If no value is returned, functions return a value of undefined.


Properties

Functions have scope, similarly to variables. Any functions written outside of a function have global scope and are available to be called, or invoked, from any where within the program. Functions written inside other functions have local scope and are only available to be called from within that function.


Examples

//script example for 1.5. Functions

alert("Let's use a function to convert kilometers to miles.");

var miles = kilo / 1.6;
