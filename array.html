1.8 Array Object

In Javascript, Arrays are objects that store multiple values in a single variable.


General Syntax

Arrays are written like this with square brackets around the list of values and commons separating the values:

var arrayName = [value1, value2, value3];

You could also use the built-in Array constructor and new key word, like this:

var arrayName = new Array(value1, value2, value3);

You can also create an array with no values, which can be added later.

Values are accessed by referencing the index number, like this:

arrayName[0]

Note that values are numbered starting at zero, so the first value in a array is index 0.


Methods

Arrays have many methods. To add new values to an array you can use the push() method, like this:

var fruit = [];

fruit.push("banana");

Another useful method is the sort() method, which allows you to put the values in alphabetical or numeric order, like this:

var vegetables = [peas, carrots, onions];

vegetables.sort();


Properties

You can use the .length property to determine how many values are contained within an array.


Examples

Here is an example of Javascript code:

//script example for 1.8 Array Object

function showExample() {

alert("Let's create an array of cheese");

var cheese = [];

cheese.push(prompt("Name one cheese"));

cheese.push(prompt("Name another cheese"));

cheese.push(prompt("Name one more cheese"));

alert("This is your array of cheese:\n" + cheese[0] + ", " + cheese[1] + ", " + cheese[2]);

cheese.sort();

alert("This is your array sorted:\n" + cheese[0] + ", " + cheese[1] + ", " + cheese[2]);
