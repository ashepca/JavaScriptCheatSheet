2.2 HTML Objects

When a web page is loaded into a browser, the HTML document becomes a document object. Each HTML tag is represented by an HTML object, for example the Anchor object represents the <a> tag, and the Form object represents the <form> tag.


General Syntax

HTML objects are part of the document object, so they are references starting with document. following by an reference to the specific HTML object, such as:

document.getElementById("idname");


Methods

There are many HTML methods available. For example, you can access HTML objects by using the getElementByID() or getElementsByName() methods, such as:

var x = document.getElementById("idname");

You can also create HTML objects by using the create(), setAttribute() and appendChild() methods, such as:

var newHTML = document.createElement("FORM");

newHTML.setAttribute("id", "aNewForm");

document.body.appendChild(newHTML);


Properties

There are many properties available. You can use properties such as .innerHTML to alter the HTML code, such as:

document.getElementById("idName").innerHTML = some other text;

Also you can use the .autocomplete property to set or return the autocomplete value of a form field, such as:

document.getElementById("idName").autocomplete = "enter name here";
