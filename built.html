1.7 JS Built-in Objects

In addition to the infinite number of objects you can create yourself, Javascript has made a number of built-in objects which are commonly used to save programming time and code. Built-in Objects work just like the objects you create except the constructor is already written.


General Syntax

There are many built-in objects in Javascript, here are a list of 5 commons ones.

  • Date()
  • Object()
  • Number()
  • Boolean()
  • Function()

Methods

  • The Date() object has methods including Date.now() which returns the current time and Date.parse() which converts a string date format into a numeric format.
  • The Object() object has methods including Object.assign() which copies values from one object to another and Object.create which create an object.
  • The Number() object has methods including Math.isNan() which determine if the value passed is a number and Math.isInteger() which determines if the value passed is an integer.
  • The Boolean() object has no metehods of its own.
  • The Function() object has no methods of its own.

Properties

  • The Date() object has properties including Date.prototype which allows you to add new properties to the Date object and Date.length which is the value of the number of arguments the constructor handles.
  • The Object() object has properties including Object.prototype which allows properties to be added and Object.length which is one.
  • The Number() object has properties including Number.MAX_VALUE and Number.MIN_VALUE which is the largest and smallest values that can be represented.
  • The Boolean() object has properties including Boolean.prototype which allows properties to be added and Boolean.length which is one.
  • The Function() object has properties including Function.name which is the name of the function and Function.length which specifies the the number of arguments.

Examples

Here is an example of Javascript code:

//script example for 1.7 JS Built-in Objects

function showExample() {

alert("Let's use a built-in object constructor to determine the date and time.");

var theDate = new Date();

alert("Today is: " + theDate);

}
