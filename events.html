2.3 Events

In Javascript, Events are things that happen that can be used to control the flow of the program. For example when the user clicks on an area of a webpage, presses a key, moves the mouse pointer over an item, or when a web page loads, these events can trigger parts of the program to execute.


General Syntax

There are many types of events. Common events include onclick, onmouseover, onload, onmouseout and onkeydown. The event is added to javascript by adding the attribute to the applicable html tag with a value that triggers javacsript code, like this:

<img onclick="myFunction()"/>

When the use clicks on the image, the javascript function myFunction() will execute.


Methods

Events do not have methods.


Properties

Events do not have properties.


Examples

Here is an example of Javascript code:

//script example for 2.3. Events

function showExample() {

alert("Click the button to get this alert was already a demonstration of an event, but let's explore events some more. Run your mouse over and out of the black circle.");

}

function growImg(that) {

that.height = (that.height * 2);

}

function shrinkImg(that) {

that.height = (that.height / 2);

}
