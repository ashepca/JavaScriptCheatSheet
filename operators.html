1.2 Operators

Javascript uses several types of operators, including Arithmetic, Assignment, Comparision and Logical. Operators 'operate' on the values you provide them. Arithmetic operators are used just like in math to perform arithmetic on numbers. Assignment operators are used to assign values to variables. Comparision and Logical operators compare values to test for true or false.


General Syntax

The Arithmetic Operators are:

  • + addition
  • - subtraction
  • * multiplication
  • / division
  • % modulus
  • ++ increment
  • ++ decrement

The Assignment Operators are:

  • = assigns a value
  • += adds and assigns
  • -= subtracts and assigns
  • *= multiplies and assigns
  • /= divides and assigns
  • %= assigns a remainder

The Comparison Operators are:

  • == equal to
  • === equal value and type
  • != not equal
  • !== not equal value or type
  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to
  • ? ternary operator

The Logical Operators are:

  • && logical and
  • || logical or
  • ! logical not

Methods

Operators have precedence. When Javascript code is exectuted operators with higher precedence are performed first. If two operators have equal precedence, they are performed left to right. Precendence can be changed by using round brackets. For example:

var pizza;

var pasta;

pizza = 2 + 2 * 7;

pasta = (2 + 2) * 7;

The value of pizza is 16 because the multiplication has higher precedence and is performed before the addition. However, pasta is 28.

Precendence for the most common used operations occurs is in this order:

  • ()
  • ++, --, !
  • *, /, %
  • +, -
  • <, <=, >, >=
  • ==, ===, !=, !==
  • &&, ||
  • all assignment operators

Properties

Operators do not have properties.
